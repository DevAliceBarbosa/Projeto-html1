Estrutura Básica do HTML

Já nessa aula, foi apresentado a sintaxe básica do HTML. A tag principal que fica envolta de todo o código é a <.html> sem o ponto depois temos as tags <.head> que seria a parte em que há algumas configurações possiveis da página, como o <.title> que fica dentro da head e serve para adicionar um titulo para a aba da página. Depois há a <.body> que fica envolta de todo o escorpo da página, ou seja, tudo que fica visivel para o usuário fica dentro dessa tag

Para fecharmos uma tag apenas é necessário colocarmos um / antes do seu nome por exemplo.

Falando sobre Tags

Nessa aula aprendemos sobre o funcionamento das Tags, que estão presentes no body Ou seja, é o que estará aparecendo para os usuários

Ele ensinou tags que alteram o texto como a <.i> que o deixa em itálico e <.strong> que deixa o texto em negrito, também podemos combiná-los de acordo com nossa preferencia.

O professor também mostrou que algumas tags se fecham nelas mesmo, como a de input para adicionar campos de interação com o usuário e img serve para adicionar imagens na página

Atributos das Tags

É exposto alguns atributos que podemos adicionar a certas tags no html

  1. Id: que serve como identificador do elemento, muito utilizado para identificá-lo no javascript
  2. Style: para adicionar uma formatação do css diretamente pelo html, como trocar cor, fonte, tamanho, borda...
  3. Class: serve para que seja criado uma classe, onde no css facilitaria para alterar as tags dessa classe de uma vez, invés de alterar manualmente cada um dos elementos
  4. type: atributo especifico da tag input, que serve para alterar o tipo do dado que o usuário poderia adicionar
  5. src: atributo da tag img, que serve para adicionar o endereço da imagem, como o caminho que deve ser percorrido até encontrar a imagem
  6. width: atributo da imagem para alterar a largura da imagem
